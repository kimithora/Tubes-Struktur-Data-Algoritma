# Project Overview

## Tujuan File
File ini menjelaskan struktur repositori dan peran setiap dokumen atau notebook, tanpa menyebutkan hasil analisis atau kesimpulan data.

## Struktur Utama
- `Churn_Modelling.csv`
  - Dataset utama yang digunakan untuk eksperimen dan analisis.

- `cek data.ipynb`
  - Notebook untuk pemeriksaan awal dan kebersihan data.
  - Menampilkan proses memuat dataset, melihat dimensi data, dan memeriksa nilai kosong.

- `EDAstd.IPYNB`
  - Notebook untuk eksplorasi statistik dasar.
  - Meliputi ringkasan distribusi data, mean, median, outlier, dan karakteristik fitur.

- `EDAAlgo.ipynb`
  - Notebook untuk eksplorasi data dan penerapan algoritma / model.
  - Dapat berisi transformasi fitur, pemodelan, dan evaluasi teknik algoritma.

- `trialerror.ipynb`
  - Notebook percobaan.
  - Digunakan untuk eksperimen, pengujian ide baru, dan iterasi.

- `README.md`
  - Panduan penggunaan proyek.
  - Menjelaskan cara membuka, menjalankan, dan memahami file-file utama.

## Cara Menggunakan File Ini
- Baca `README.md` jika Anda ingin tahu cara mengoperasikan proyek.
- Baca `PROJECT_OVERVIEW.md` jika Anda ingin memahami struktur dan tujuan setiap file.
- Gunakan notebook untuk melihat implementasi teknis secara langsung.

## Catatan
- `PROJECT_OVERVIEW.md` bukan ringkasan hasil analisis.
- File ini hanya menjelaskan organisasi konten dan peran tiap berkas.
